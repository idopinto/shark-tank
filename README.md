# Requirment
  * npm
  
# Usage

Clone it!
```
git clone https://github.com/idopinto/shark-tank.git
```

Go into the project directory and run the command:
```
cd shark-tank
```
then, run the command:
```
npm install
npm audit fix
npm install react-is
```
to install node modules.

Finally, run the command:
```
npm start
```
Enjoy ;)
